什么是SVG

  1. SVG 指可伸缩矢量图形(Scalable Vector Graphics),一种二维图形表示语言
  2. SVG 用来定义用于网络的基于矢量的图形
  3. SVG 使用 XML 格式定义图形
  4. SVG 图像在放大或改变尺寸的情况下其图形质量不会有所改变
  5. SVG 是万维网联盟的标准
  6. SVG 与诸如 DOM 和 XSL 之类的 W3C 标准是一个整体

SVG的优势

  1. SVG 可被非常多的工具读取和修改
  2. SVG 与 JPG 和 GIF 图像比起来，文件更小，且可压缩性更强
  3. SVG 是可伸缩的
  4. SVG 图像可在任何分辨率下被高质量的打印
  5. SVG 能在图像质量不下降的情况下放大
  6. SVG 图像中的文本是可选的，也是可搜索的
  7. SVG 可以与java一起运行
  8. SVG 是开放的标准
  9. SVG 文件是纯粹的XML

SVG 示例

在页面中添加svg

<body>
    <svg height="100"></svg></body>

定义矩形

x="20" y="20"表示矩形的起点为(20,20)

stroke:边框颜色 stroke-width:边框宽度

<rect x="20" y="20" width="50" height="50" stroke="blue" stroke-width="5" />

定义圆形

r 表示圆的半径

cx="110" cy="45"表示矩形的起点为(110,45)

<circle cx="110" cy="45" r="25" stroke="red" stroke-width="5" fill="#fff" />

定义文本

<text x="150" y="50" font-family="microsoft yahei" font-size="20px" fill="green">SVG 示例</text>

定义线性渐变

<linearGradient id="Gradient">
    <stop offset="0%" stop-color="red" />
    <stop offset="100%" stop-color="blue" />
</linearGradient>
<rect x="20" y="80" width="50" height="50" fill="url(#Gradient)" />
