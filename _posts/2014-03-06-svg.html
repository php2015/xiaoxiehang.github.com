什么是SVG

  1. SVG 指可伸缩矢量图形(Scalable Vector Graphics),一种二维图形表示语言
  2. SVG 用来定义用于网络的基于矢量的图形
  3. SVG 使用 XML 格式定义图形
  4. SVG 图像在放大或改变尺寸的情况下其图形质量不会有所改变
  5. SVG 是万维网联盟的标准
  6. SVG 与诸如 DOM 和 XSL 之类的 W3C 标准是一个整体

SVG的优势

  1. SVG 可被非常多的工具读取和修改
  2. SVG 与 JPG 和 GIF 图像比起来，文件更小，且可压缩性更强
  3. SVG 是可伸缩的
  4. SVG 图像可在任何分辨率下被高质量的打印
  5. SVG 能在图像质量不下降的情况下放大
  6. SVG 图像中的文本是可选的，也是可搜索的
  7. SVG 可以与java一起运行
  8. SVG 是开放的标准
  9. SVG 文件是纯粹的XML

SVG 示例


在页面中添加svg

内联方式：像HTML中的其它元素一样使用，在此基础上可以编写HTML，JavaScript和SVG的交互应用

<body>
    <svg height="100"></svg>
</body>

外联方式：通过<img>元素，在HTML导入外部SVG文件，缺点是无法编写与SVG元素进行交互的脚本。

<img src="example.svg" />

定义矩形

x="20" y="20"表示矩形的起点为(20,20)

stroke:边框颜色 stroke-width:边框宽度

<rect x="20" y="20" width="50" height="50" stroke="blue" stroke-width="5" />

圆形

r 表示圆的半径

cx="110" cy="45"表示矩形的起点为(110,45)

<circle cx="40" cy="40" r="25" stroke="red" stroke-width="5" fill="#fff" />

文本

<text x="20" y="40" font-family="microsoft yahei" font-size="20px" fill="green">SVG 示例</text>

渐变

<linearGradient id="Gradient">
    <stop offset="0%" stop-color="red" />
    <stop offset="100%" stop-color="blue" />
</linearGradient>
<rect x="20" y="40" width="50" height="50" fill="url(#Gradient)" />

图案

<pattern id="Pattern" patternContentUnits="userSpaceOnUse" x="0" y="0" width="100" height="67" viewBox="0 0 100 67">
    <image x="0" y="0" width="80" height="80" xlink:href="http://xiaojjblog.qiniudn.com/face.jpg" />
</pattern>
<circle cx="40" cy="40" r="25" stroke="red" stroke-width="5" fill="url(#Pattern)" />

组

可以用来结合多个相关的元素，并且可以作为一个整体进行变换（包括变形、旋转、缩放等）。组内成员可以通过ID来引用。

<g transform="translate(100,0) rotate(30) scale(0.5)" id="ShapeGroup">
    <rect x="20" y="20" width="50" height="50" stroke="blue" stroke-width="5" />
    <circle cx="110" cy="45" r="25" stroke="red" stroke-width="5" fill="#fff" />
    <text x="150" y="50" font-family="microsoft yahei" font-size="20px" fill="green">SVG 示例</text>
</g>

复用内容

<defs>元素用于定义留待将来使用的内容。可以用<use>元素将<defs>定义的内容链接到需要展示的地方。借助这两个元素，你可以多次复用同一内容，消除冗余。

<defs>
    <g id="defs">
        <rect x="20" y="20" width="50" height="50" stroke="blue" stroke-width="5" />
        <circle cx="110" cy="45" r="25" stroke="red" stroke-width="5" fill="#fff" />
        <text x="150" y="50" font-family="microsoft yahei" font-size="20px" fill="green">SVG 示例</text>
    </g>
</defs>
<use xlink:href="#defs" />
<use xlink:href="#defs" transform="translate(260,0) scale(0.75)" />
<use xlink:href="#defs" transform="translate(460,0) scale(0.5)" />
