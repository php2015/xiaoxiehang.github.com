什么是SVG

  1. SVG 指可伸缩矢量图形(Scalable Vector Graphics)
  2. SVG 用来定义用于网络的基于矢量的图形
  3. SVG 使用 XML 格式定义图形
  4. SVG 图像在放大或改变尺寸的情况下其图形质量不会有所改变
  5. SVG 是万维网联盟的标准
  6. SVG 与诸如 DOM 和 XSL 之类的 W3C 标准是一个整体

SVG的优势

  1. SVG 可被非常多的工具读取和修改
  2. SVG 与 JPG 和 GIF 图像比起来，文件更小，且可压缩性更强
  3. SVG 是可伸缩的
  4. SVG 图像可在任何分辨率下被高质量的打印
  5. SVG 能在图像质量不下降的情况下放大
  6. SVG 图像中的文本是可选的，也是可搜索的
  7. SVG 可以与java一起运行
  8. SVG 是开放的标准
  9. SVG 文件是纯粹的XML
